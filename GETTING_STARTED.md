## ğŸ“ Key Concepts Cheat Sheet

### React
- **Component** = Reusable UI piece
- **Props** = Data passed to component
- **State** = Data that can change
- **useState** = Hook to manage state
- **useEffect** = Hook to run code on load/update

### Spring Boot
- **@RestController** = Creates API endpoint
- **@Service** = Business logic layer
- **@Autowired** = Dependency injection
- **GET/POST** = HTTP methods
- **JSON** = Data format

### APIs
- **REST** = API design style
- **Endpoint** = URL path (e.g., `/api/chat`)
- **Request** = What you send
- **Response** = What you receive
- **CORS** = Cross-origin security
